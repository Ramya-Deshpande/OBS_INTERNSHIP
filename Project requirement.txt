Title: Applying feature engineering techniques to dataset.

Objective:
The objective of this project is to: 
Ensure data quality by removing errors, inconsistencies, and missing values.
Standardize numerical and categorical data formats for better interpretability.
Detect and handle outliers to improve data reliability.
Convert data types to appropriate formats for analysis and modeling.
Remove duplicate records and irrelevant columns to optimize data storage.
Provide a structured, clean dataset suitable for machine learning and statistical analysis.

Requirements:
1.Python 3.x
2.Pandas
3.Numpy
4.Matplotlib
5.Jupyter Notebook
